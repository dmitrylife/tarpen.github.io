# Tarpen — общественный проект

Публичная платформа для фиксации и решения инфраструктурных проблем в посёлке Тушино (Калининградская область).

## Цель проекта

- фиксация проблем (дороги, мосты, доступность, экология);
- хранение доказательств (фото, видео, документы);
- создание прозрачной хронологии событий;
- публичный контроль и взаимодействие с органами власти.

## Разделы

- /docs/bridge — аварийное состояние мостового сооружения
- /docs/road-access — нарушение транспортной доступности
- /docs/trees — незаконная вырубка деревьев

## Принципы

- только факты и подтверждённые данные;
- открытость и прозрачность;
- отсутствие политизации;
- защита интересов жителей.

## Материалы

Все материалы (фото, видео, обращения) публикуются для фиксации фактов и общественного контроля.
